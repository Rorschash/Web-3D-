Web 3D Apps is a final year and postgraduate module . This is my first web 3D application. It build an X3D based Coca Cola Branded (3D App) exploiting HTML5, CSS3, X3D (X3DOM), JavaScript, and a JavaScript frontend framework(Bootstrap), AJAX for asynchronous update of HTML contents, JSON for data transfer from the backend to the frontend. I used JavaScript to control the display and animations of the 3D model.
